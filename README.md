# Kotach Engineering

Marketing site for Kotach Engineering — civil and structural engineering for UK rail infrastructure.

Static site: `index.html` + `style.css` + `script.js`. Deployed on Vercel as static output (no build step).

## Local preview

Any static server works:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

Vercel auto-deploys on push to the connected branch. No framework, no build command.
